# Cara Belanja di Hoky

Temukan produk di halaman hoky.id

![website hoky](../../.gitbook/assets/image%20%28220%29.png)

Pilih produk yang ingin di beli, dapat dilakukan dengan menggunakan fitur pencarian.

![Halaman pencarian hoky](../../.gitbook/assets/image%20%28219%29.png)

  
Kemudian pilih product yang akan dilihat, kemudian akan muncull detail dari produk.

![](../../.gitbook/assets/image%20%2817%29.png)

Kemudian Anda klik tombol **Buy Now** untuk memasukan produk ke keranjang belanja dan melanjutkan pencarian produk yang lain. \(untuk proses pembelian diharuskan melakukan login terlebih dahulu\)

Pilih varian jika produk memiliki variant, dan tentukan jumlah item produk yang ingin di beli, tekan add to cart untuk memasukan pesanan ke kerancang belanja Anda

tekan cart untuk melakukan checkout pembelian

![](../../.gitbook/assets/image%20%2828%29.png)

![](../../.gitbook/assets/image%20%28165%29.png)

![](../../.gitbook/assets/image%20%2829%29.png)
